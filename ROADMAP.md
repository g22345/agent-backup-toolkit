# Roadmap

The roadmap is directional, not a delivery promise.

## v0.1 alpha

- Finish packaging and cross-platform CI.
- Run the synthetic backup/verify/restore drill with the real `age` binary.
- Complete public-tree and Git-history review.
- Recruit a small number of genuine external testers after publication approval.

## Possible later work

- Signed receipts or another sender-authentication mechanism.
- More granular restore selection.
- Additional well-reviewed agent-workspace presets.
- Provider contract tests against disposable private test infrastructure.
- Retention policy only after a safe preview and recovery design exists.

Out of scope for v0.1: hosted dashboards, plaintext backups, shell hooks,
automatic credential discovery, public GitHub destinations, and target deletion.
